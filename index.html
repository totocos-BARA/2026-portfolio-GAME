[ 01 — ABOUT ]

게임 안에서 살아 움직이는
캐릭터를 만듭니다.

학창시절부터 일본 애니메이션과 서브컬처를 좋아해 미소녀 캐릭터 그리기에 빠졌습니다. "내가 그린 캐릭터가 게임 속에서 살아 움직이고, 유저들이 사랑해준다"는 것 — 그게 제 게임아트의 출발점입니다.

펀게임을 시작으로 그라비티(섬란카구라), 라이언게임즈(소울워커), 웹젠노바(테르비스), 넥슨코리아(메이플스토리)를 거치며 모바일 · 온라인 라이브 서비스 · 글로벌 IP 협업을 두루 경험. 캐릭터 원화 · 일러스트 · 도트 · 이펙트 · UI 아이콘 · Spine 파츠 분리까지 게임아트 풀 파이프라인을 다룹니다.

원작사 검수가 까다로운 마벨러스(섬란카구라) · 토에이(로봇걸즈) IP 협업 경험과, 700명 규모 본부(메이플 스튜디오)에서 인게임 비주얼을 다뤄본 양쪽 환경 모두에서 일해본 게 강점입니다.

[ 04 — AI WORKFLOW ]

AI로 가속하는
게임아트 파이프라인.

직접 그린 베이스 컨셉을 ChatGPT · Gemini · Stable Diffusion 등 AI 도구로 디포메이션 · 자세 변형 · 다양화한 뒤, Photoshop · Procreate 로 리터칭. 하나의 캐릭터를 빠르게 여러 용도로 확장하는 워크플로우를 운영합니다. 게임 업계에서도 AI 활용이 빠르게 자리잡는 흐름에 맞춰 실무에 바로 적용 가능한 파이프라인입니다.

STEP 01

베이스 컨셉 직접 작업

Procreate / Photoshop 으로 캐릭터 컨셉·키 비주얼을 직접 그림. 아트의 정체성은 사람이 만듭니다.

STEP 02

AI로 변형 · 확장

ChatGPT / Gemini 로 디포메이션, 자세 변형, 표정 시트, 다양한 의상 안 생성. 속도와 아이디어는 AI가 더합니다.

STEP 03

리터칭으로 마감

디테일·라인·컬러를 사람이 다시 손봐 통일된 톤으로 마감. 완성도는 사람이 책임집니다.

// IMPACT 하나의 캐릭터 컨셉을 디포메이션·표정 시트·이벤트 일러스트 등 5~10가지 변형으로 확장하는 데 걸리던 시간을 1/3~1/5 로 줄였습니다. 작가의 손맛은 유지하면서, 빠르게 IP 활용 폭을 넓힐 수 있는 게 강점입니다.
